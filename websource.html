Introduction

Project Goal

The "Boat Racing" game was built for entertainment purposes, while helping each member of the group learn and program to get acquainted with how to use the Pygame library to develop games. The game provides an exciting and challenging experience for players as they control the boat to avoid obstacles on the race track.

Scope

  • Folder /img: Where to store images for the game
  • Folder /wav: Where to store audio data played in the game
  • Folder /src: Where to store the main source code to design the game, including:
    • main.py: Main source code running the game
    • config.py: General configuration of the project (default parameters)
    • background.py: Manage and display the background
    • boat.py: Manage the movement and display control of the boat
    • obstacles.py: Manage the movement, display and collision check of obstacles
    • buffitem.py: Manage the movement, display and collision check of power-up items
    • hearth.py: Manage the durability status of the boat
    • score.py: Manage game points
    • sound.py: Manage sound

Rules of the game

The player controls the boat moving on the background, avoiding obstacles that appear randomly on the road in order to survive as long as possible. The longer you survive, the faster the boat will be based on the score you have achieved. Along the way, you can pick up power-up items to increase your survivability.

Main features

  • Control the boat with the arrow keys (left, right, up, down).
  • Obstacles and power-up items appear randomly and move down from the top of the screen.
  • Sound enhances the vibrancy of the game.
  • Scoring system.
  • Start and end game interface.

Difficulties in implementation

  • Difficulty in designing the interface.
  • Some functions have errors, so I have to use ChatGPT software to refer.
  • Time issues so it is not possible to develop many other functions.
